# 音声まとめ (Audio Summarizer)

動画・音声ファイルを入力するだけで、文字起こし・要約・箇条書き・キーワード抽出を行うデスクトップアプリです。

## 特徴

- **ローカル完結** — API課金なし、インターネット接続不要（初回モデルダウンロード除く）
- **ドラッグ＆ドロップ対応** — ファイルを置くだけで処理開始
- **日本語最優先** — 日本語音声の文字起こしに最適化
- **多形式対応** — mp4, mov, m4a, mp3, wav, avi, mkv, webm, ogg, flac, aac, wma
- **保存機能** — txt / md / json で結果をエクスポート

## 技術スタック

| レイヤー | 技術 | 採用理由 |
|---------|------|---------|
| UI | Electron + React + TypeScript | クロスプラットフォーム、安定性 |
| ビルド | electron-vite | HMR対応の高速開発環境 |
| 音声抽出 | FFmpeg (fluent-ffmpeg) | バイナリ同梱でセットアップ不要 |
| 文字起こし | @xenova/transformers (Whisper ONNX) | npm installのみで完結、Python不要 |
| テキスト処理 | 自作 TF-IDF抽出型エンジン | 外部API不要、即座に結果 |

### Whisper ONNX を採用した理由

- `npm install` だけで導入完了（Python/C++ビルド環境不要）
- ブラウザ向けONNXランタイムで動作するため環境依存が少ない
- 初回はモデルをダウンロード（約500MB）、以降はキャッシュ利用
- whisper-small モデルで日本語の実用的な精度を確保

## セットアップ

```bash
cd claud/audio-summarizer
npm install
```

## 起動（開発モード）

```bash
npm run dev
```

## ビルド（配布用）

```bash
npm run build
npm run package
```

`dist/` フォルダにインストーラーが生成されます。

## 使い方

1. アプリを起動
2. 動画または音声ファイルをドラッグ＆ドロップ（またはファイル選択ボタン）
3. 自動で処理開始（初回はAIモデルのダウンロードに数分かかります）
4. 結果がタブ表示される
   - **文字起こし** — タイムスタンプ付き全文
   - **要約** — 短い/標準/詳細の3段階
   - **箇条書き** — 重要ポイント5〜10個
   - **キーワード** — 重要語句（スコア付き）
5. コピーボタンまたは保存ボタンで結果を取得

## 対応ファイル形式

| 種類 | 形式 |
|------|------|
| 動画 | .mp4, .mov, .avi, .mkv, .webm |
| 音声 | .m4a, .mp3, .wav, .ogg, .flac, .aac, .wma |

## 処理フロー

```
ファイル選択 → 音声抽出(FFmpeg) → 文字起こし(Whisper) → テキスト分析 → 結果表示
```

## 注意事項

- 初回起動時にWhisperモデル（約500MB）をダウンロードします
- 長い動画（1時間超）は処理に時間がかかります
- メモリは4GB以上推奨
- 元ファイルは一切変更しません（読み取りのみ）
