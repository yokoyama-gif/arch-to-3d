# PDF Utility Pro

住宅・建設・事務現場向けの Windows デスクトップ PDF 編集ユーティリティです。
ドラッグ＆ドロップでPDFを読み込み、ページの並べ替え・削除・回転・結合・抽出を1画面で行えます。

---

## 主な機能

| 機能 | 説明 |
|---|---|
| **PDF読み込み** | ドラッグ＆ドロップ または ファイルダイアログ。複数ファイル一括対応 |
| **サムネイル一覧** | 全ページをカード形式で表示。読み込み後に非同期レンダリング |
| **並べ替え** | サムネイルをドラッグして順番を変更。「上へ」「下へ」ボタンも使用可 |
| **削除** | 選択ページを削除。Ctrl+Z で元に戻せる |
| **回転** | 選択ページを左右90°回転。複数同時対応 |
| **全体を保存** | 現在の並び順・回転を反映した1つのPDFとして保存 |
| **選択ページを保存** | 選択したページだけを新しいPDFとして抽出・保存 |
| **複数選択** | Ctrl+クリック、Shift+クリック（範囲）、Ctrl+A（全選択）|
| **他形式に変換** | PDF→画像（PNG/JPEG、解像度選択可）・PDF→テキスト（.txt）に変換 |
| **画像→PDF** | PNG/JPEG画像をPDFに変換して取り込み（D&Dでも可） |

## 技術スタック

- **フレームワーク**: [Electron](https://www.electronjs.org/) 28
- **UI**: [React](https://react.dev/) 18 + [TypeScript](https://www.typescriptlang.org/) 5
- **ビルドツール**: [electron-vite](https://electron-vite.org/)
- **PDF生成・編集**: [pdf-lib](https://pdf-lib.js.org/)
- **PDFレンダリング（サムネイル）**: [pdfjs-dist](https://github.com/mozilla/pdfjs-dist) 3.x
- **ドラッグ＆ドロップ**: [@dnd-kit](https://dndkit.com/)

## セットアップ方法

### 前提条件

- **Node.js** 18 以上（推奨: 20 LTS）
- **npm** 9 以上（Node.js に同梱）

### インストール

```bash
# リポジトリをクローン（またはフォルダをコピー）した後、プロジェクトに移動
cd pdf-utility-pro

# 依存関係のインストール
npm install
```

---

## 起動方法（開発）

```bash
npm run dev
```

Electron ウィンドウが起動します。  
DevTools は `Ctrl+Shift+I` で開けます（開発モードのみ）。

---

## ビルド方法

### アプリケーションのビルド（配布用バイナリ生成）

```bash
npm run dist
```

`dist/` フォルダに Windows 向けインストーラー（`.exe`）が生成されます。

### ソースのコンパイルのみ（インストーラー不要）

```bash
npm run build
```

`out/` フォルダにコンパイル済みファイルが生成されます。

## 注意点

- **ローカル完結**: ファイルはすべてローカルで処理されます。クラウド送信はありません
- **上書き保存なし**: 保存は常に「名前を付けて保存」ダイアログを使用します
- **日本語パス対応**: 日本語のファイル名・フォルダ名に対応しています
- **暗号化PDF**: パスワード付きPDFは読み込めない場合があります
- **大容量PDF**: 100ページを超える場合はサムネイル生成に時間がかかります

## 操作一覧

### マウス操作

| 操作 | 動作 |
|---|---|
| クリック | そのページのみ選択 |
| **Ctrl+クリック** | 選択の追加／解除（複数選択） |
| **Shift+クリック** | 直前選択から今クリックしたページまで範囲選択 |
| **ダブルクリック** | 全画面で拡大表示 |
| ドラッグ | サムネイルをドラッグして並べ替え |
| PDFをドロップ | ウィンドウにドラッグ＆ドロップで追加 |

### キーボード（通常画面）

| キー | 動作 |
|---|---|
| `Ctrl+A` | 全ページ選択 |
| `Ctrl+Z` | 元に戻す（最大20ステップ） |
| `Delete` | 選択ページを削除 |
| `F1` / `?` | 操作ヘルプを表示 |

### キーボード（拡大表示中）

| キー | 動作 |
|---|---|
| `←` / `→` | 前後のページへ移動 |
| `L` | 左に90°回転 |
| `R` | 右に90°回転 |
| `Delete` | 表示中のページを削除 |
| `Esc` | 拡大表示を閉じる |

アプリ画面右上の「❓ 操作ヘルプ」ボタン、または `F1` キーでも同じ一覧を表示できます。
